2. THE "GOLDEN FORMULA" (REQUIREMENTS)
--------------------------------------
- Minimum 8 Unique Sections per page.
- No content duplication between landing pages.
- Rhythmic color flow: Alternating Dark (#1A2D42) and Light (White) sections.
- Decorative Grid Pattern: Mandatory SVG overlay on all dark #1A2D42 backgrounds.
- High Button Contrast: 
    - Dark backgrounds -> Silver buttons ($wirtschaft-silver).
    - Light backgrounds -> Navy buttons ($wirtschaft-primary).
- Mobile Optimization: 
    - Section padding reduced from 120px to 60px.
    - Heading (h2) size reduced from 3rem to 2rem.
    - Grid gaps reduced to 40px for narrow screens.
- Global Synchronization: Use identical Header, Footer, and "Aktuelles" (Blog) blocks.
